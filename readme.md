## INSTRUCTION

How to run:

```
git clone git@github.com:fastindian84/walking_bot.git

cd walking_bot

bundle install

bin/start

```

Initial commands:

```
PLACE X,Y,F
MOVE
LEFT
RIGHT
REPORT
```

Additional commands:

`clear` - clear history

`pretty` - display the board

`exit` - exit the program


-----

`rspec` to run tests
